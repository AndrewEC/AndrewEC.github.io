Steganography App

A command line utility demonstrating the concept of steganography by hiding and retrieving data in images by applying changes to the least significant bit (LSB) of each RGB value in the image's pixels.

  • Can hide a single large file in multiple images
  • Encrypts the file's contents using AES-256 before hiding it
  • Minifies the file's contents using standard GZip compression
  • Randomize the file's byte order before storing it
  • Clean function to clear out any saved data on one or more images that currently contain data
